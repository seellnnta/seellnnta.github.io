Pelajaran 04

List

List mewakili urutan bernilai di Python. Modul ini mencakup pembuatan list, indexing, slicing, mutasi, fungsi bawaan, metode, pencarian, dan perbandingan dengan tuple.

Mengenal List

List dibuat memakai kurung siku dan dapat berisi angka, string, maupun list lain.

primes = [2, 3, 5, 7]
planets = ['Mercury', 'Venus', 'Earth', 'Mars', 'Jupiter', 'Saturn', 'Uranus', 'Neptune']
hands = [
    ['J', 'Q', 'K'],
    ['2', '2', '2'],
    ['6', 'A', 'K'],
]
my_favourite_things = [32, 'raindrops on roses', help]

Indexing

Akses elemen tunggal dengan indeks berbasis nol. Gunakan indeks negatif untuk menghitung dari belakang.

planets[0]   # 'Mercury'
planets[1]   # 'Venus'
planets[-1]  # 'Neptune'
planets[-2]  # 'Uranus'

Slicing

Slicing menggunakan notasi list[start:end] dan end tidak termasuk.

planets[0:3]   # ['Mercury', 'Venus', 'Earth']
planets[:3]    # ['Mercury', 'Venus', 'Earth']
planets[3:]    # ['Mars', 'Jupiter', 'Saturn', 'Uranus', 'Neptune']
planets[1:-1]  # ['Venus', 'Earth', 'Mars', 'Jupiter', 'Saturn', 'Uranus']
planets[-3:]   # ['Saturn', 'Uranus', 'Neptune']

start atau end boleh dikosongkan dan Python akan menganggapnya awal/akhir list.

Mengubah List (Mutable)

List bisa diubah di tempat. Penugasan ke indeks atau slice akan memodifikasi nilai.

planets[3] = 'Malacandra'
planets[:3] = ['Mur', 'Vee', 'Ur']
print(planets)

# Kembalikan ke semula
planets[:4] = ['Mercury', 'Venus', 'Earth', 'Mars']

Fungsi Bawaan Berguna

  • len(planets) menghitung elemen
  • sorted(planets) mengembalikan list baru yang terurut
  • sum(primes) menjumlahkan angka
  • min(primes) dan max(primes) mencari nilai ekstrem
len(planets)      # 8
sorted(planets)   # Urutan alfabetis
primes = [2, 3, 5, 7]
sum(primes)       # 17
max(primes)       # 7

Sekilas mengenai Objek

Semua di Python adalah objek lengkap dengan atribut dan metode, diakses lewat tanda titik.

x = 12
print(x.imag)           # 0 (bagian imajiner)

c = 12 + 3j
print(c.imag)           # 3.0

print(x.bit_length())   # 4
help(x.bit_length)

Metode adalah fungsi yang menempel pada objek (mis. bit_length pada integer).

Metode List

List memiliki metode seperti append dan pop untuk memodifikasi isi.

planets.append('Pluto')
print(planets)
# ['Mercury', 'Venus', 'Earth', 'Mars', 'Jupiter', 'Saturn', 'Uranus', 'Neptune', 'Pluto']

planets.pop()
print(planets)
# ['Mercury', 'Venus', 'Earth', 'Mars', 'Jupiter', 'Saturn', 'Uranus', 'Neptune']

help(planets.append)

Mencari dalam List

Pakai index untuk menemukan posisi elemen dan operator in untuk pengecekan keberadaan.

planets.index('Earth')          # 2
'Earth' in planets              # True
'Calbefraques' in planets       # False
⚠️ Perhatian: planets.index('Pluto') akan memunculkan ValueError karena Pluto tidak lagi di list.

Tuples

Tuple mirip list tetapi memakai kurung biasa dan immutable (tidak bisa diubah).

t = (1, 2, 3)
t = 1, 2, 3      # Kurung opsional

# t[0] = 100     # TypeError: tuple tidak mendukung item assignment

x = 0.125
num, den = x.as_integer_ratio()
print(num / den)  # 0.125

# Swap variables dengan tuple
a, b = 1, 0
a, b = b, a
print(a, b)  # 0 1

Tuple sering digunakan untuk mengembalikan beberapa nilai dari satu fungsi.

Ringkasan

List adalah struktur data fundamental di Python yang mutable dan mendukung indexing, slicing, serta berbagai metode untuk manipulasi data. Tuple memberikan alternatif immutable yang berguna untuk data yang tidak boleh berubah.
