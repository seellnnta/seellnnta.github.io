Halo, Python

Pelajaran pembuka ini memperkenalkan sintaks Python, cara kerja variabel, operator aritmetika, dan fitur-fitur dasar lain yang akan sering kamu jumpai saat menganalisis data.

Gambaran Kursus

Kursus Python Kaggle dirancang untuk pegiat data yang sudah pernah menulis kode di bahasa lain dan ingin menambahkan Python ke kotak alat mereka. Kita mulai dengan sintaks inti, assignment variabel, lalu masuk ke operasi matematika dan fungsi bawaan.

Jika kamu benar-benar baru dalam dunia koding, pertimbangkan untuk mengikuti kursus Intro to Programming terlebih dahulu sebelum melanjutkan.

Program Pertama: Homage Monty Python

Baca kode berikut, tebak apa yang terjadi, lalu jalankan di notebook dan perhatikan output-nya.

spam_amount = 0
print(spam_amount)

# Ordering Spam, egg, Spam, Spam, bacon and Spam (4 more servings of Spam)
spam_amount = spam_amount + 4

if spam_amount > 0:
    print("But I don't want ANY spam!")

viking_song = "Spam " * spam_amount
print(viking_song)

Kode lucu ini sudah memanfaatkan banyak konsep kunci Python. Mari kita bedah baris demi baris.

Assignment Variabel

Pada baris spam_amount = 0 kita membuat variabel tanpa perlu mendeklarasikan tipe. Python bersifat dynamically typed, sehingga variabel bisa menyimpan angka, string, bahkan boolean secara bergantian.

Memanggil print(spam_amount) akan menampilkan 0. Fungsi print() dipanggil dengan menuliskan nama fungsi diikuti kurung dan argumen.

Komentar & Penugasan Ulang

Baris yang diawali # adalah komentar, berguna untuk menjelaskan maksud kode. Setelahnya kita melakukan penugasan ulang:

spam_amount = spam_amount + 4

Python mengevaluasi bagian kanan (0 + 4) lalu menyimpan hasilnya ke variabel yang sama.

Conditionals & Blok Kode

Percabangan ditulis dengan if diakhiri titik dua. Setiap baris yang diberi indentasi empat spasi setelahnya dianggap bagian dari blok tersebut.

if spam_amount > 0:
    print("But I don't want ANY spam!")

Pada bahasa lain kamu mungkin menemui kurung kurawal; Python menggantinya dengan indentasi yang konsisten sehingga kode lebih mudah dibaca.

String & Operator *

String dapat dikalikan dengan angka untuk mengulang teks:

viking_song = "Spam " * spam_amount
print(viking_song)
# Output: Spam Spam Spam Spam

Ini contoh operator overloading, di mana operator yang sama memiliki makna berbeda bergantung pada tipenya.

Angka & Fungsi type()

Gunakan type() untuk memeriksa tipe data:

spam_amount = 0
type(spam_amount)   # int
type(19.95)         # float

int mewakili bilangan bulat, sementara float menyimpan angka dengan bagian desimal.

Operator Aritmetika

Python mendukung seluruh operator kalkulator di bawah ini:

Operator Nama Deskripsi
a + b Penjumlahan Menjumlahkan a dan b
a - b Pengurangan Selisih a dan b
a * b Perkalian Hasil kali
a / b Pembagian Hasil bagi (float)
a // b Floor division Hasil bagi dibulatkan ke bawah
a % b Modulus Sisa bagi
a ** b Pangkat a dipangkatkan b
-a Negasi Mengubah tanda angka

/ selalu menghasilkan float, sedangkan // memberikan bilangan bulat yang dibulatkan ke bawah, berguna misalnya saat membagi dataset ke batch.

Prioritas Operasi

Python mengikuti aturan serupa PEMDAS. Gunakan tanda kurung jika ingin mengontrol urutan evaluasi.

hat_height_cm = 25
my_height_cm = 190
total_height_m = (hat_height_cm + my_height_cm) / 100
print("Height in meters =", total_height_m)

Pada contoh di atas, menambahkan kurung memastikan penjumlahan dilakukan sebelum pembagian.

Fungsi Bawaan untuk Angka

  • min() dan max() mencari nilai terkecil dan terbesar.
  • abs() mengembalikan nilai mutlak.
  • int() dan float() dapat mengonversi angka maupun string numerik.
print(min(1, 2, 3))      # 1
print(abs(-32))         # 32
print(int('807') + 1)   # 808
