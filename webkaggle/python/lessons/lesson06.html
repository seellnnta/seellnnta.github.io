String & Dictionary

Manipulasi teks dan pasangan kunci-nilai adalah senjata utama saat membersihkan data, membangun fitur, ataupun berdialog dengan API.

Anatomi String Python

String bisa ditulis memakai kutip tunggal maupun ganda, hasilnya sama saja.

x = 'Pluto is a planet'
y = "Pluto is a planet"
print(x == y)  # True
print("Pluto's a planet!")
print('My dog is named "Pluto"')

Jika perlu menaruh kutip tunggal di dalam kutip tunggal, pakai karakter escaping \.

'Pluto\'s a planet!'
"Pluto's a planet!"

Escape Sequence Penting

Yang diketik Hasil Contoh Output
\' ' 'What\'s up?' What's up?
\" " "That\"s cool" That's "cool"
\\ \ "Look, a mountain: /\\" Look, a mountain: /\
\n Newline "1\n2 3" 1
2 3

Triple quote memungkinkan memasukkan newline secara literal.

hello = "hello\nworld"
print(hello)

triple = """hello
world"""
print(triple)
print(triple == hello)  # True

print() menambahkan newline otomatis kecuali kita ganti parameter end.

print("hello")
print("world")
print("hello", end='')
print("pluto", end='')

String sebagai Urutan

Semua operasi list seperti indexing, slicing, dan iterasi berlaku untuk string, namun string tidak bisa dimodifikasi inplace.

planet = 'Pluto'
print(planet[0])      # 'P'
print(planet[-3:])    # 'uto'
print(len(planet))    # 5
print([c + '! ' for c in planet])

# planet[0] = 'B'  # TypeError: string is immutable

Metode String Populer

claim = "Pluto is a planet!"
print(claim.upper())      # PLUTO IS A PLANET!
print(claim.lower())      # pluto is a planet!
print(claim.index('plan'))  # 12
print(claim.startswith('Pluto'))  # True
print(claim.endswith('planet'))   # False (karena ada !)

split() & join()

str.split() memecah string menjadi list, sedangkan str.join() menyatukan list menjadi string.

claim = "Pluto is a planet!"
words = claim.split()
print(words)  # ['Pluto', 'is', 'a', 'planet!']

datestr = '1956-01-31'
year, month, day = datestr.split('-')
print('/'.join([month, day, year]))  # 01/31/1956

separator = ' clap '.join([w.upper() for w in words])
print(separator)  # PLUTO clap IS clap A clap PLANET!

Menyusun String dengan format()

str.format() memberi placeholder {} yang otomatis mengubah tipe data ke string.

planet = 'Pluto'
position = 9
print("{}, you'll always be the {}th planet to me.".format(planet, position))

pluto_mass = 1.303 * 10**22
earth_mass = 5.9722 * 10**24
population = 52_910_390
msg = "{} weighs about {:.2} kilograms ({:.3%} of Earth's mass). It is home to {:,} Plutonians.".format(
    planet,
    pluto_mass,
    pluto_mass / earth_mass,
    population,
)
print(msg)

s = """Pluto's a {0}.
No, it's a {1}.
{0}!
{1}!""".format('planet', 'dwarf planet')
print(s)

Dictionary 101

Dictionary memetakan kunci ke nilai menggunakan sintaks kurung kurawal.

numbers = {'one': 1, 'two': 2, 'three': 3}
print(numbers['one'])  # 1

numbers['eleven'] = 11
numbers['one'] = 'Pluto'
print(numbers)  # {'one': 'Pluto', 'two': 2, 'three': 3, 'eleven': 11}

Gunakan dict.get() untuk nilai default saat kunci tidak ada.

print(numbers.get('missing', 'Tidak tersedia'))  # Tidak tersedia

Dictionary Comprehension & Keanggotaan

planets = ['Mercury', 'Venus', 'Earth', 'Mars', 'Jupiter', 'Saturn', 'Uranus', 'Neptune']
planet_to_initial = {planet: planet[0] for planet in planets}
print(planet_to_initial)

print('Saturn' in planet_to_initial)      # True
print('Betelgeuse' in planet_to_initial)  # False

Iterasi atas Dictionary

Loop standar berjalan pada kunci; gunakan .items() untuk mendapatkan pasangan kunci-nilai sekaligus.

numbers = {'one': 'Pluto', 'two': 2, 'three': 3, 'eleven': 11}
for key in numbers:
    print(f"{key} = {numbers[key]}")

for planet, initial in planet_to_initial.items():
    print(f"{planet.rjust(10)} begins with \"{initial}\"")

initials = ' '.join(sorted(planet_to_initial.values()))
print(initials)  # E J M M N S U V

Ringkasan

String memberi alat kaya untuk formatting, pencarian, dan pemisahan teks, sedangkan dictionary menyimpan data berlabel serta mendukung komprehensi, iterasi, dan pengecekan kunci cepat. Kuasai keduanya untuk mempermudah parsing dataset dunia nyata.
