Pustaka Eksternal

Import membuka akses ke pustaka standard maupun pihak ketiga seperti math, numpy, pandas, dan tensorflow. Pelajaran ini memandu sintaks import, submodul, dan cara membaca objek asing.

Mengapa Import?

Selain tipe bawaan, Python punya ribuan library. Sebagian berada di standard library, sisanya dipasang lewat pip. Semuanya diakses melalui kata kunci import.

import math

print("It's math! It has type {}".format(type(math)))

Output menunjukkan bahwa math adalah modul (namespace berisi variabel dan fungsi).

Menyelami Modul math

Lihat seluruh nama yang tersedia dengan dir(), lalu akses via dot notation.

print(dir(math))
print("pi ke 4 digit signifikan = {:.4}".format(math.pi))
print(math.log(32, 2))  # 5.0
help(math.log)

help(math) menampilkan dokumentasi gabungan untuk seluruh anggota modul.

Sintaks Import Alternatif

Alih nama modul agar lebih singkat atau impor simbol spesifik.

import math as mt
print(mt.pi)

from math import log, pi
print(pi, log(32, 2))

from math import *
from numpy import *
# print(pi, log(32, 2))  # TypeError: fungsi log numpy menimpa versi math

Submodul dan Namespace Bersarang

Modul dapat berisi modul lain. Contoh, numpy.random adalah submodul.

import numpy
print(type(numpy.random))
print(dir(numpy.random)[-10:])

rolls = numpy.random.randint(low=1, high=6, size=10)
print(rolls)

Pemanggilan butuh dua titik: numpy.random.randint.

Panduan Mengenal Objek Aneh

Gunakan tiga fungsi bawaan untuk memahami tipe baru: type(), dir(), dan help().

type(rolls)          # numpy.ndarray
print(dir(rolls))
print(rolls.mean())
print(rolls.tolist())
help(rolls.ravel)

help(rolls) menampilkan dokumentasi lengkap mengenai ndarray.

Operator Overloading

Tipe custom dapat mendefinisikan ulang operator.

# [3, 4, 1] + 10  # TypeError
rolls + 10      # setiap elemen ditambah 10

rolls <= 3      # menghasilkan array boolean

xlist = [[1, 2, 3], [2, 4, 6]]
x = numpy.asarray(xlist)
print(x[1, -1])  # 6
# print(xlist[1, -1])  # TypeError: list tidak mendukung tuple indexing

ndarray mendukung indexing multi dimensi karena perancangnya memang meng-overload operator terkait.

Kejutan TensorFlow

import tensorflow as tf

a = tf.constant(1)
b = tf.constant(1)
print(a + b)
# Hasilnya bukan angka 2, melainkan tf.Tensor

Hasilnya bukan angka 2, melainkan tf.Tensor yang merupakan handle simbolik ke operasi penjumlahan. Banyak library (termasuk pandas) memanfaatkan operator overloading secara kreatif.

# Contoh di pandas (pseudo)
df[(df['population'] > 10**6) & (df['continent'] == 'South America')]

Ekspresi tersebut mengandalkan beberapa operator yang telah dioverride.

Memahami Nama Magic

dir(list) menampilkan banyak metode dengan double underscore seperti __contains__ atau __add__. Metode inilah yang dipanggil Python saat Anda memakai operator.

print('__contains__' in dir(list))  # True
print([1, 2, 3].__contains__(2))    # True, ekuivalen dengan 2 in [1, 2, 3]

Mengetahui konvensi ini membantu saat membaca dokumentasi tipe khusus.

üí° Tips Praktis: Tiga fungsi untuk eksplorasi library:
  • type(obj) - Mengetahui tipe objek
  • dir(obj) - Melihat atribut dan metode yang tersedia
  • help(obj) - Membaca dokumentasi lengkap

Ringkasan

Gunakan import untuk membuka kemampuan baru, pahami namespace modul dengan dir() dan help(), berhati-hatilah terhadap from module import *, dan ingat bahwa operator bisa berperilaku beda pada tiap tipe. Dengan pemahaman ini, Anda akan lebih percaya diri menjelajah ekosistem Python yang luas.
